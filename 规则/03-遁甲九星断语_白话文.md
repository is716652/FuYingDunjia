## 一、九星基础：天上的九位“能量指挥官”

奇门遁甲认为，宇宙中有九颗主星（对应北斗七星加辅弼二星），它们轮流值班，影响人间事务。每颗星固定镇守一个宫位（九宫），代表不同性质的能量。

| 九星 | 宫位 | 八卦 | 方位 | 五行 | 吉凶分类（基础） |
| ---- | ---- | ---- | ---- | ---- | ---------------- |
| 天蓬 | 一宫 | 坎 ☵ | 正北 | 水   | **大凶**         |
| 天芮 | 二宫 | 坤 ☷ | 西南 | 土   | **大凶**         |
| 天冲 | 三宫 | 震 ☳ | 正东 | 木   | **次吉**         |
| 天辅 | 四宫 | 巽 ☴ | 东南 | 木   | **上吉**         |
| 天禽 | 五宫 | 中   | 中央 | 土   | **上吉**         |
| 天心 | 六宫 | 乾 ☰ | 西北 | 金   | **上吉**         |
| 天柱 | 七宫 | 兑 ☱ | 正西 | 金   | **小凶**         |
| 天任 | 八宫 | 艮 ☶ | 东北 | 土   | **次吉**         |
| 天英 | 九宫 | 离 ☲ | 正南 | 火   | **小凶**         |

> 💡 **吉凶等级说明**：
>
> - **上吉**：天辅、天禽、天心 → 百事皆宜，贵人扶持  
> - **次吉**：天冲、天任 → 有特定用途，需看时机  
> - **大凶**：天蓬、天芮 → 多主灾病、盗贼、失败  
> - **小凶**：天英、天柱 → 有偏性，用之不当则凶

## 二、关键原则：吉凶不是固定的！要看“旺相休囚”

古人强调：**星星的能量会随季节变化而强弱不同**。
即使“大凶之星”，若在旺季，也可能转为“小凶”；
而“上吉之星”，若在衰季，也可能只算“中平”。

### 2.1 九星旺衰规则（核心！）

> **口诀**：
> “九星各旺于我生月，相于同类月，死于生我月，囚于官鬼月，休于才月。”

✅ 白话解释（以**天蓬水星**为例）：

| 状态   | 月份（地支）               | 五行关系            | 能量强度      |
| ------ | -------------------------- | ------------------- | ------------- |
| **旺** | 寅、卯月（春）             | 水生木 → 我生者旺   | ⭐⭐⭐⭐⭐（最强） |
| **相** | 亥、子月（冬）             | 水+水 → 同类相助    | ⭐⭐⭐⭐          |
| **死** | 巳、午月（夏）             | 火生土 → 生我者耗我 | ⭐（极弱）     |
| **囚** | 辰、戌、丑、未月（四季月） | 土克水 → 官鬼克制   | ⭐             |
| **休** | 申、酉月（秋）             | 金生水 → 被生而安   | ⭐⭐            |

> 📌 **通用规则表**（适用于所有星）：

| 星的五行                     | 旺（我生月）   | 相（同类月）   | 死（生我月）   | 囚（克我月）   | 休（我克月）   |
| ---------------------------- | -------------- | -------------- | -------------- | -------------- | -------------- |
| **木星**（天冲、天辅）       | 巳午（火）     | 寅卯（木）     | 亥子（水）     | 申酉（金）     | 辰戌丑未（土） |
| **火星**（天英）             | 辰戌丑未（土） | 巳午（火）     | 寅卯（木）     | 亥子（水）     | 申酉（金）     |
| **土星**（天芮、天禽、天任） | 申酉（金）     | 辰戌丑未（土） | 巳午（火）     | 寅卯（木）     | 亥子（水）     |
| **金星**（天心、天柱）       | 亥子（水）     | 申酉（金）     | 辰戌丑未（土） | 巳午（火）     | 寅卯（木）     |
| **水星**（天蓬）             | 寅卯（木）     | 亥子（水）     | 申酉（金）     | 辰戌丑未（土） | 巳午（火）     |

> ✅ **应用口诀**：
> “**大凶星得旺相 → 小凶；小凶星得旺相 → 中平；上吉星无气 → 中平**”
> → 务必结合季节动态判断！

#### ▶ 实例

> 冬至后，得天任星（土星）。
> 冬属水（亥子月）→ 土星处“死地”（水克土）→ 本应不吉。
> 但原文说：“假令冬至后时下得天任宿，吉；乘旺相之气，为上吉也。”  
>
> ❓矛盾吗？
> ✅ 不矛盾！因“冬至后”虽在冬季，但**阳气初生，艮土（天任）得令**，且天任为“次吉星”，在特定节气可转旺。
> → 说明：**节气交接时，旺衰需精细判断**。

## 三、九星详细解读：能做什么？不能做什么？

以下按星逐一详解，包含：**宜做之事、忌做之事、季节吉凶、实战提示**。

------

### 3.1 天蓬星（一宫·坎·水）——“冒险家/盗贼星”

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 智慧、冒险、流动性强，但也主盗贼、破财、血光                 |
| **宜做**     | - 安抚边境 - 修筑城池（利用其“防御”属性） - 春夏季节出兵（左将大胜） |
| **忌做**     | - 秋冬用事（凶亡） - 嫁娶、搬家、入官、经商 - 争斗、见血、防火灾 |
| **季节吉凶** | - **春（旺）**：可用，利军事 - **夏（相）**：次吉 - **秋（囚）**：大凶 - **冬（死）**：极凶 |
| **特殊提示** | “其士卒利主不利客” → 防守方有利，进攻方不利                  |

> 🚨 **总结**：非军事用途慎用！普通人遇天蓬，多主破财、被骗、意外。

------

### 3.2 天芮星（二宫·坤·土）——“病星/学生星”

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 疾病、问题、学习、拜师                                       |
| **宜做**     | - 求医问药（反向利用） - 拜师学艺 - 结交朋友（祟尚修道）     |
| **忌做**     | - 用兵、嫁娶、诉讼、搬家、盖房 - **春秋用事大凶**            |
| **季节吉凶** | - **秋冬（旺/相）**：吉（土生金，金生水？实为土旺于四季，但古法以秋冬为吉） - **春夏（囚/死）**：大凶 |
| **特殊提示** | 天芮为“病星”，但若求医或解决难题，反可借其“聚焦问题”之力     |

> 💡 **妙用**：测病时，天芮落宫即病灶方位；求学时，天芮为“老师星”。

------

### 3.3 天冲星（三宫·震·木）——“雷神/行动星”

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 冲动、出击、雷电、快速行动                                   |
| **宜做**     | - 出兵报仇 - 春夏季节作战（左将胜）                          |
| **忌做**     | - 秋冬用事（无功） - 嫁娶、搬家、入官、祭祀、做生意          |
| **季节吉凶** | - **春（旺）**：大吉 - **夏（相）**：吉 - **秋（囚）**：凶 - **冬（死）**：大凶 |
| **特殊提示** | 只适合“快进快出”的短期行动，不适合长期经营                   |

------

### 3.4 天辅星（四宫·巽·木）——“文曲星/贵人星”（上吉）

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 教育、文化、贵人、道德                                       |
| **宜做**     | - 读书学习 - 设立学校 - 带兵（春夏胜） - 嫁娶（多子多福） - 搬家、经商、入官、盖房（春夏尤吉） |
| **忌做**     | 无明确禁忌，但秋冬力量减弱                                   |
| **季节吉凶** | - **春（旺）**：大吉 - **夏（相）**：吉 - **秋（囚）**：中平 - **冬（死）**：中平 |
| **特殊提示** | “得平地千里” → 行动顺利，阻力小                              |

> ✅ **万能吉星**，尤其利文化、教育、人际关系。

------

### 3.5 天禽星（五宫·中·土）——“帝王星/调和星”（上吉）

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 中正、稳定、百事皆宜                                         |
| **宜做**     | - 祭祀求福 - 带兵（四时皆吉，敌人畏服） - 封官授爵 - 搬家、入官、祭祀、经商、嫁娶 |
| **忌做**     | 几乎无禁忌                                                   |
| **季节吉凶** | **四季皆吉**（中央土，不随季节衰旺）                         |
| **特殊提示** | “百福助之” → 天时地利人和齐聚                                |

> 👑 **最尊贵之星**，遇之必吉，无需看季节！

------

### 3.6 天心星（六宫·乾·金）——“医星/领导星”（上吉）

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 医疗、科技、领导力、制度                                     |
| **宜做**     | - 治病合药 - 带兵（秋冬胜） - 嫁娶、入官、盖房、祭祀、经商（秋冬吉） - 君子行事 |
| **忌做**     | - 春夏用事不利 - 小人行事（反招祸）                          |
| **季节吉凶** | - **秋（旺）**：大吉 - **冬（相）**：吉 - **春（囚）**：凶 - **夏（死）**：大凶 |
| **特殊提示** | “得地千里” → 掌控全局，资源丰富                              |

> 🩺 **医疗、管理首选星**，但务必选对季节。

------

### 3.7 天柱星（七宫·兑·金）——“破坏星/隐士星”（小凶）

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 口舌、破坏、变革、隐藏                                       |
| **宜做**     | - 屯兵自守 - 隐居避世 - 嫁娶、修造、祭祀（有限可用）         |
| **忌做**     | - 带兵出征（车破马伤，士卒败亡） - 搬家、入官、经商          |
| **季节吉凶** | - **秋（旺）**：小凶（破坏力强） - **冬（相）**：中平 - **春（囚）**：凶 - **夏（死）**：大凶 |
| **特殊提示** | 利守不利攻，宜静不宜动                                       |

------

### 3.8 天任星（八宫·艮·土）——“诚信星/建筑星”（次吉）

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 诚信、稳重、建筑、积累                                       |
| **宜做**     | - 拜访贵人 - 通财利 - 带兵（四时皆吉，敌人自降） - 嫁娶（利子孙） - 入官 |
| **忌做**     | - 搬家、盖房（原文明确忌“移徙、筑室”）                       |
| **季节吉凶** | - **四季月（旺）**：大吉 - **秋冬（相/休）**：吉 - **春夏（囚/死）**：中平 |
| **特殊提示** | “万神助之” → 得天地助力，但仅限特定事项                      |

> ❓为何宜“入官”却忌“筑室”？
> ✅ 因天任主“守成”，适合任职（稳定），但“筑室”需变动，与其“止”性冲突。

------

### 3.9 天英星（九宫·离·火）——“名声星/急躁星”（小凶）

| 项目         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **基本象意** | 名声、急躁、火灾、虚花                                       |
| **宜做**     | - 出行、远行 - 饮宴作乐 - 嫁娶（利名声）                     |
| **忌做**     | - 出兵、搬家、入官、盖房、诉讼、经商                         |
| **季节吉凶** | - **夏（旺）**：小凶（火炎土燥） - **春（相）**：中平 - **秋（囚）**：凶 - **冬（死）**：大凶 |
| **特殊提示** | 利“虚名”不利“实利”，适合社交娱乐，不适合务实操作             |

## 四、出行呼神法（古法秘术）

古人出行前，会呼唤当值星神的名字以求庇护。

| 九星 | 神字（呼名） | 出行方向       |
| ---- | ------------ | -------------- |
| 天蓬 | 子禽         | 正北           |
| 天芮 | 子成         | 西南           |
| 天冲 | 子翘         | 正东           |
| 天辅 | 子卿         | 东南           |
| 天禽 | 子公         | 中央（或任意） |
| 天心 | 子襄         | 西北           |
| 天柱 | 子申 / 子常  | 正西           |
| 天任 | 子韦 / 子金  | 东北           |

> 📌 **用法示例**：
> 冬至上元阳遁一局，丙寅时，欲出东北（艮方），此时天蓬直符加八宫 → 呼“**子禽**”，行60步，入太阴方位，可避凶趋吉。

> 💡 现代意义：可作为心理暗示或仪式感设计，增强决策信心。

## 五、总结：九星使用心法

1. **先看吉凶分类**：上吉星优先，大凶星慎用。
2. **再看季节旺衰**：吉星无气 = 中平；凶星得旺 = 小凶。
3. **结合具体事项**：同一星，对不同事吉凶不同（如天芮利求医不利嫁娶）。
4. **勿孤立判断**：需结合八门、神煞、格局综合分析。

> ✅ **终极口诀**：
> “**吉星得令百事顺，凶星失时不为殃；
> 若问用事宜与忌，须看星门共神章。**”
