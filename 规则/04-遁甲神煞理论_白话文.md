## 一、核心神煞体系：九天、九地、太阴、六合

这四大神煞是奇门遁甲中**最重要的空间能量分区**，用于判断在某方位行动是否吉利。

### 1.1 基础定义（张良曰）

| 神煞     | 象意                             | 宜用场景               |
| -------- | -------------------------------- | ---------------------- |
| **九天** | 至高无上的阳刚之气，如天空般开阔 | 利于排兵布阵、主动出击 |
| **九地** | 至深至厚的阴柔之气，如大地般沉稳 | 利于埋伏隐藏、长期潜伏 |
| **太阴** | 阴柔隐秘之神，主谋略、暗中行动   | 利于逃亡、秘密行动     |
| **六合** | 和合之神，主合作、婚姻、交易     | 利于谈判、签约、嫁娶   |

> ✅ **口诀记忆**：
> “**九天陈兵，九地伏藏，太阴六合，可以逃亡**”

------

### 1.2 定位方法：以“直符”为基准

神煞位置**不是固定的**，而是随“天上直符”所在宫位动态变化。

#### ▶ 冬至后（阳遁）——顺时针排布

假设直符在 **X 宫**，则：

| 神煞 | 相对位置               | 实际宫位 |
| ---- | ---------------------- | -------- |
| 九天 | 直符 **后一宫**（逆1） | X - 1    |
| 九地 | 直符 **后二宫**（逆2） | X - 2    |
| 太阴 | 直符 **前二宫**（顺2） | X + 2    |
| 六合 | 直符 **前三宫**（顺3） | X + 3    |

> 📌 注：宫位按洛书顺序：1→2→3→4→5→6→7→8→9→1（循环）

#### ▶ 夏至后（阴遁）——逆时针排布

假设直符在 **X 宫**，则：

| 神煞 | 相对位置               | 实际宫位 |
| ---- | ---------------------- | -------- |
| 九天 | 直符 **前一宫**（顺1） | X + 1    |
| 九地 | 直符 **前二宫**（顺2） | X + 2    |
| 太阴 | 直符 **后二宫**（逆2） | X - 2    |
| 六合 | 直符 **后三宫**（逆3） | X - 3    |

> 🔢 **计算规则**：  
>
> - 宫位超出9 → 减9；低于1 → 加9  
> - 中五宫（5）通常寄坤二宫（西南）

------

### 1.3 实例演示

#### ▶ 阳遁上元一局（冬至后）

- 时间：甲己日丙寅时  
- 直符临 **八宫**（艮，东北）  
- 阳遁 → 用“后一、后二、前二、前三”规则：

| 神煞 | 计算           | 宫位 | 方位 |
| ---- | -------------- | ---- | ---- |
| 九天 | 8 - 1 = 7      | 七宫 | 正西 |
| 九地 | 8 - 2 = 6      | 六宫 | 西北 |
| 太阴 | 8 + 2 = 10 → 1 | 一宫 | 正北 |
| 六合 | 8 + 3 = 11 → 2 | 二宫 | 西南 |

✅ 结论：此时**正西利出兵，西北利埋伏，正北利逃亡，西南利谈判**。

#### ▶ 阴遁上元九局（夏至后）

- 时间：甲己日丙寅时  
- 直符临 **二宫**（坤，西南）  
- 阴遁 → 用“前一、前二、后二、后三”规则：

| 神煞 | 计算           | 宫位 | 方位 |
| ---- | -------------- | ---- | ---- |
| 九天 | 2 + 1 = 3      | 三宫 | 正东 |
| 九地 | 2 + 2 = 4      | 四宫 | 东南 |
| 太阴 | 2 - 2 = 0 → 9  | 九宫 | 正南 |
| 六合 | 2 - 3 = -1 → 8 | 八宫 | 东北 |

✅ 结论：此时**正东利进攻，东南利潜伏，正南利秘密行动，东北利合作**。

## 二、六甲、六癸、六丁、六己：四大能量符号

这四组“六仪”不仅是排盘基础，更代表特定时空能量。

### 2.1 六甲 = 九天之上（统帅之位）

- **含义**：六甲（甲子、甲戌…甲寅）为“直符”，象征最高指挥官。

- **原理**：
  《易经》说“乾纳甲壬”，乾为天，天道上升。从壬数到甲，共9数 → 故称“九天之上”。

- **应用**：

  > “士众常背直符所临之宫，而击其冲，无不胜也。”
  > → 军队应背靠直符方，攻击对面方向（冲宫），必胜！

> 💡 **现代引申**：做决策时，以“直符方”为靠山（如办公室坐向），面向对宫行动。

------

### 2.2 六癸 = 九地之下（隐遁之位）

- **含义**：六癸（癸酉、癸未…癸亥）为“九地之下”，主隐藏、消失。

- **原理**：
  《易经》说“坤纳乙癸”，坤为地，地道下降。从乙数到癸，共9数 → 故称“九地之下”。

- **应用**：

  > “凡逃亡绝迹，当以天上六癸所临之方下出入。”
  > → 逃亡、躲债、避灾时，从**六癸落宫方位**进出，无人能见。

> 🕵️ **操作步骤**：  
>
> 1. 找出天盘六癸落在哪一宫  
> 2. 从该方位悄悄出入  
> 3. 配合“甸甸而去”（低头快走）仪式

------

### 2.3 六丁 = 三奇之灵（护身之神）

- **含义**：六丁（丁卯、丁丑…丁巳）为“星奇”，最具灵性。
- **神名与字**（呼名护身）：

| 丁干 | 神名 | 神字 | 应用   |
| ---- | ---- | ---- | ------ |
| 丁卯 | 文伯 | —    | 主文书 |
| 丁丑 | 文孙 | —    | 主子孙 |
| 丁亥 | 文公 | —    | 主长辈 |
| 丁酉 | 文通 | —    | 主沟通 |
| 丁未 | 文卿 | —    | 主贵人 |
| 丁巳 | 巨卿 | —    | 主权威 |

> ✅ **口诀**：
> “能知六丁出幽冥，至老不刑；刀虽临颈，犹安不惊。”
> → 懂得用六丁的人，一生平安，临危不惧。

> 🙏 **用法**：
> 凡斗争、出行、办事，心中默念或轻呼对应“神字”，可得护佑。

------

### 2.4 六己 = 六合之中（私密之门）

- **含义**：六己（己巳、己卯…己丑）为“六合之中”，主秘密、隐私。

- **原理**：
  甲与己合 → 故称“六合”。举甲子旬为例，故统称“六己巳”。

- **应用**：

  > “凡为阴谋、密秘、伏隐之事，皆从天上六己所临之方而出，人莫见之。”
  > → 做机密事（如商业谈判、秘密约会），从**六己落宫方位**出发，无人察觉。

> 🚪 **关键句**：
> “六合为私门，独出独入，无有见者。”

## 五、特殊时空理论

### 5.1 威德之时（六丙为威，六甲为德）

- **构成**：六丙（威） + 六甲（德）

- **效果**：  

  > “入其国，犬不吠，马不嘶……敢有举兵来向者，皆还自灭。”
  > → 威德并施，不战而屈人之兵。

- **应用**：  

  - 利为“客”（主动方）  
  - 不利为主（防守方）  
  - 宜固守待时，不宜先发

------

### 5.2 之三避五（生气 vs 害气）

- **三** = 生气（吉） → “之三” = 趋向三宫  
- **五** = 害气（凶） → “避五” = 避开五宫（中宫）
- **原理**：
  “三为威，五为武，盛于三，衰于五。”
- **实例**：  
  - 冬至上元阳一局，丙寅时 → 宫得3 → 百事吉  
  - 戊辰时 → 宫得5 → 百事凶

> ✅ **操作**：重要事情尽量安排在“三宫值时”，避开“五宫值时”。

------

### 5.3 天网四张（极端凶时）

- **触发条件**：时干为 **六癸**

- **含义**：  

  > “天网四张，万物尽伤。此时不可举造百事。”

- **避法**：  

  1. 找到“天乙”（即值符）所在宫  
  2. 低头（甸句）  
  3. 以左右肩纽面前行过十步  
  4. 若天乙高二尺以上，可灵活避让

> 🚫 **结论**：六癸时，万事暂停！

------
