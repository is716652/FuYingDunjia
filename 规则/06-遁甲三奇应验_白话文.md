## 一、什么是“三奇”？

在奇门遁甲中，“三奇”是最吉利的三个天干，代表宇宙精华能量：

| 三奇   | 名称 | 象意                         | 能量属性 |
| ------ | ---- | ---------------------------- | -------- |
| **乙** | 日奇 | 太阳之精，主文书、贵人、柔顺 | 木（阴） |
| **丙** | 月奇 | 月亮之华，主光明、权威、变动 | 火（阳） |
| **丁** | 星奇 | 北斗星光，主智慧、灵性、突发 | 火（阴） |

> ✅ **核心原则**：  
>
> - 三奇需配合**吉门（开、休、生）** 才能发挥最大效力；  
> - 若落**凶宫、入墓、被克**，则吉力减弱甚至转凶。

## 二、三奇运行规则：阳遁 vs 阴遁

- **阳遁**（冬至后）：三奇**顺行**（从一宫→九宫）  
- **阴遁**（夏至后）：三奇**逆行**（从九宫→一宫）

> 📌 **案例说明**：
> 阳一局，甲午日直符在三宫 → 顺取用 → 丙奇（月奇）在四宫 → “月奇到巽（四宫），应时中”。

## 六、奇门路应：出门遇事预兆

在路上遇到特定人物或事件，可判断所办事宜吉凶。

| 三奇     | 门类 | 路上所见                | 吉凶 | 应验事项         |
| -------- | ---- | ----------------------- | ---- | ---------------- |
| **乙奇** | 生门 | 两只老鼠打架 / 穿孝服者 | 吉   | 事有竞争，终得财 |
|          | 休门 | 牛马 / 扛木头的人       | 吉   | 得助力，事缓成   |
|          | 开门 | 客人 / 红衣官吏         | 吉   | 官非得解，贵人助 |
| **丙奇** | 生门 | 凶眼人 / 打架者         | 凶   | 争斗破财         |
|          | 休门 | 50里内闻鼓声或乐器      | 吉   | 喜庆将至         |
|          | 开门 | 老人执杖 / 哭声         | 凶   | 事有阻碍         |
| **丁奇** | 生门 | 猪或狗出现              | 吉   | 得小财           |
|          | 休门 | 20里内遇穿黑/白衣妇人   | 吉   | 阴人助力         |
|          | 开门 | 小孩执竹杖              | 吉   | 子女/晚辈助成事  |

> 🚶 **使用场景**：
> 出门前或途中观察，可即时调整策略。

## 七、总结：三奇应验心法

### 7.1 吉应共性

- **人物**：多为特定衣着、职业、数量的人  
- **动物**：鸟（双至为吉）、牛马、猪犬  
- **声音**：鼓、乐、雷、啼哭  
- **财物**：金银、契约、田地、活物  
- **时间**：短则当日，长则1–3年

### 7.2 凶应警示

- 多与“死亡、火灾、雷击、自缢”相关  
- 但常为“应期触发点”，非单纯凶兆

### 7.3 实用建议

1. **记录奇仪落宫**：排盘后重点看乙、丙、丁在哪一宫  
2. **结合方位行动**：如丁奇在兑（西），可向西求财  
3. **等待应期**：按“1/7/100日”或“周年”耐心等待  
4. **勿孤立判断**：需结合八门、格局、旺衰综合分析

> ✅ **终极口诀**：
> “**三奇到宫，万象昭彰；
> 人物声色，皆为吉殃；
> 时至事应，不可彷徨。**”
