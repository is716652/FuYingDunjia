# 三遁·地遁规则

## 一、原文定位

出自《御制景祐遁甲符应经·中卷》“地遁”条：
- “地遁者，开门与六乙日奇合临地下六己，为地遁。此时得月精之蔽也。”
- 并举阳一局、丙辛日的具体例子说明盘面组合。

## 二、规则要点

1. **核心构成要素**
   - 门：**开门**
   - 奇：**六乙（日奇）**
   - 地干：**六己（六合/地户）**
   - 三者在特定宫位形成合局，即为“地遁”。
2. **本质含义**
   - “得月精之蔽”
   - 可理解为：行动得到“地利 / 环境 / 资源”的庇护，更利于**布局型、建设型、资源配置型**活动。

## 三、判定条件（工程化版本）

1. **必要条件**
   - 至少存在一宫满足：
     - 门 = 开门；
     - 天盘奇 = 乙（即“六乙日奇”）；
   - 且与“六己所在宫位”构成原文所述的“临地下六己”关系（可由引擎根据天盘/地盘建模进一步精细化）。

2. **可简化判定版本（第一阶段实现建议）**
   - 在未实现完整“六合/地盘”建模前，可用：
     - 若某宫 **门 = 开门** 且 **天盘天干 = 乙**，则在UI上标记为：
       - “**地遁候选格**（开门遇日奇乙）”
     - 高级说明中提示：
       - “是否为完全天遁，仍需结合六己位置进一步判断（高级规则）”。

## 四、吉凶与应用建议

1. **适宜事项**
   - 资源调度、基础建设、内部布局类活动：
     - 如古法中的营造、筑城、整备后勤、安营扎寨等
   - 在现代研习语境中，可对应：
     - 规划性工作、基础设施搭建、环境/系统性调整
   - 三奇择日页可给出建议文案：
     - “**地遁：利于布局与基础建设，适合先稳固根基。**”

2. **不宜事项**
   - 需要极高机动性、强对抗性的正面冲突（这类更偏向天遁/利客格局去研习）

## 五、在APP中的推荐呈现方式

1. **格局标签**
   - 名称：**“地遁”**
   - 分类：高级格局 · 三遁之一

2. **提示文案（示例）**
   - 简短版：
     - “开门遇日奇六乙并得六合之助，为地遁格，得月精之蔽，宜布局与建设。”
   - 展开版：
     - “依据《遁甲符应经》中‘地遁’规则，本盘开门与日奇乙奇合，兼得六合六己之助，为地遁格局。象征地利与环境配合，适合研习基础建设、内部布局等主题。”

## 六、与天遁、人遁的对比

- **天遁**：生门 + 丙奇 + 六丁 → 偏“天时”、主动性强
- **地遁**：开门 + 乙奇 + 六己 → 偏“地利”、重在布局
- **人遁**：休门 + 丁奇 + 太阴 → 偏“人和/隐秘行动”

在UI层推荐：
- 在同一位置（如格局总览列表）将三者并列展示，便于研习者对比“天时 / 地利 / 人和”的不同应用场景。
