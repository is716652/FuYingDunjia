# 三奇入墓规则

## 一、原文定位

出自《御制景祐遁甲符应经·中卷》“三奇入墓”条：
- 葛洪曰：
  - “三奇入墓者，谓乙未时为乙奇入墓，乙为日奇，木墓在未，故谓日奇入墓之时；丙戌为月奇，火墓在戌，故谓月奇入墓。”
- 玉璋曰：
  - “三奇墓者，谓六乙日奇临二宫，六丙月奇，六丁星奇在六宫，是谓三奇入墓也。”

## 二、规则要点

1. **三奇**
   - 乙：日奇
   - 丙：月奇
   - 丁：星奇
2. **“入墓”含义**
   - 依五行墓库（木墓在未，火墓在戌等）判断奇所在的“墓位”，象征：
     - 三奇的力量**被收束、潜伏，难以外显**。
3. **两个层面**
   - **时间层面**：特定时辰（如乙未、丙戌）
   - **盘局层面**：六乙临二宫，六丙/六丁在六宫的盘面结构

## 三、判定条件（工程化版本）

### 1. 时间维度判定

- 若当前**时辰干支**满足：
  - 乙未 → 判定为“日奇入墓时”；
  - 丙戌 → 判定为“月奇入墓时”；
- 可在整体择日/用时建议中提示：
  - “当前为日奇/月奇入墓时段，三奇之力偏向收敛，不宜过分指望奇门之奇。”

### 2. 盘面维度判定

- 若排盘结果中满足：
  - 六乙（日奇）在二宫；
  - 六丙（月奇）、六丁（星奇）在六宫；
- 则此盘形成“**三奇入墓格**”。
- 可视为：三奇整体进入墓库，象征**奇用受限、潜伏不显**。

## 四、吉凶与应用建议

1. **整体倾向**
   - 三奇入墓 ≠ 单纯大凶，但表示：
     - 奇门中的“奇”难以外展，更适合**收敛、结束、反思、内在整理**类行为。
2. **适宜事项（研习向）**
   - 做阶段性收尾、复盘、整理工作
   - 进行不需大张旗鼓的内向型安排（内部调整、旧事了结）
3. **不宜事项**
   - 过分依赖“奇门奇迹”的激进行动
   - 期待一蹴而就的大突破

## 五、在APP中的推荐呈现方式

1. **格局标签**
   - 名称：**“三奇入墓”**
   - 类型：高级格局 · 三奇状态

2. **提示文案（示例）**
   - 简短版：
     - “三奇入墓：奇气收敛，宜收尾整顿，不宜激进突破。”
   - 展开版：
     - “依据《遁甲符应经》中‘三奇入墓’规则，本盘三奇（乙、丙、丁）落入墓库位置，象征奇门之奇收束潜伏，更适合研习收尾、整理、内省型主题，谨慎看待激进行动。”

## 六、与三奇得使、三奇之灵的关系

- 在整体三奇体系中：
  - “三奇得使 / 三奇之灵” → 三奇被充分调动、效果显著；
  - “三奇入墓” → 三奇暂时隐伏、难以发挥；
- 在未来格局总览中，可将三奇相关若干规则合成一组“**三奇状态总览**”：
  - 当前是“得使 / 得令 / 入墓 / 平常”中的哪一种，以便研习者一眼看到三奇状态。
