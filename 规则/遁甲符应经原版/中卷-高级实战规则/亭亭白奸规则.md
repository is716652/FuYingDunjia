# 亭亭白奸规则

## 一、原文定位

出自《御制景祐遁甲符应经·中卷》“亭亭白奸”两段：
- “玉璋曰：亭亭者，天之贵神，背而击其冲，为胜推之法。常以月将加时，神后下为亭亭所居也。”
- “白奸者，天之奸神，合于亥，格于寅申，当合之时俱背之，当格与不格，合与不合者，背亭亭向白奸，推之法：以月将加时，寅午戌上见孟神，即是白奸之位，常以行寅申己亥四孟位也。”

## 二、规则要点

1. **亭亭**
   - 属性：天之贵神
   - 形成方式：
     - 以**月将加时**，神后下为亭亭所居
   - 用法：
     - “背而击其冲，为胜推之法” → 背对亭亭位置，向其冲击方向用兵，可得胜机
2. **白奸**
   - 属性：天之奸神
   - 形成方式：
     - 以月将加时，于寅午戌上见孟神，即白奸位
     - 行寅申己亥四孟位
   - 用法：
     - “背亭亭向白奸”，即在有亭亭、白奸两点时，选定“背贵趋奸”的方向推测胜负。

## 三、判定条件（工程化版本，建议偏研习解释型）

> 亭亭/白奸规则高度依赖“月将加时”“孟神”等天盘系统，短期更适合作为**知识展示**而非精确算法。

1. **亭亭定位（高阶预留）**
   - 以“月将加时”，神后下为亭亭所居：
     - 需引擎提供：月将位置、神后位置、加时后的落点
2. **白奸定位（高阶预留）**
   - 以月将加时，在寅午戌上见孟神，即为白奸位：
     - 需引擎具备“月将+十二神+孟仲季”完整体系

在当前工程阶段，建议仅在 UI 中：
- 提供“亭亭/白奸”概念说明与简单示意，不做强制自动判宫。

## 四、吉凶与应用建议（偏兵法象义，供研习）

1. **亭亭（贵神）**
   - 背而击其冲 → 象征：
     - 以贵神为依托，从其冲方向发力，可获胜机
2. **白奸（奸神）**
   - “背亭亭向白奸”
     - 在部分用兵推演中，用于**判断敌方薄弱或可攻之位**

## 五、在 APP 中的推荐呈现方式

1. **知识卡片型呈现**
   - 在高级规则说明页中，增加“亭亭与白奸”一卡：
     - 简要原文 + 白话解释 + 示意图（后续可加）。
2. **暂不参与自动格局计算**
   - 现阶段不建议在引擎中硬编码亭亭/白奸判定，待后续若引入月将体系，再行结构化。
