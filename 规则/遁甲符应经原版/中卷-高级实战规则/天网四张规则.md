# 天网四张规则

## 一、原文定位

出自《御制景祐遁甲符应经·中卷》“天网四张”条：
- “三元经曰：天网四张，万物尽伤。此时不可举造百事。”
- 并进一步说明：
  - “谓时下浮六癸之时……”
  - 以及天乙所在宫高度与“匍匐而去”的避凶法。

## 二、规则要点

1. **基本含义**
   - “天网四张，万物尽伤”：
     - 象征上天之网张开，万物易受损伤
     - 此时不宜举造百事，为**强凶时段**
2. **触发条件（原文）**
   - 关键条件为：**“时下浮六癸之时”**，即时刻与六癸相关联的特殊状态。
   - 同时提到：天乙所在宫高度不同，对避凶方式有差异。

## 三、判定条件（工程化版本）

1. **简化判定建议（优先级高）**
   - 由于原文对“时下浮六癸”的算法依赖更完整的天盘/地盘与六十甲子布局，第一阶段建议：
     - 以“当前时干 = 癸”或“时干对应六癸浮临”为触发近似条件；
     - 一旦触发，即在 UI 中提示“天网四张强凶警告”。
2. **严格判定预留**
   - 后续若引擎实现了：
     - 六十甲子旬首与六癸浮临计算
     - 天乙所在宫、高度（去地一尺、二尺等）的建模
   - 可根据完整“时下浮六癸 + 天乙高度”精细化判断，决定是否必须“匍匐而去”以避网。

## 四、吉凶与应用建议

1. **整体倾向**
   - 强凶规则：
     - “不可举造百事” → 不宜启动任何重要事务
   - 更适合作为“全局时段的强烈减分项”。
2. **研习向解释**
   - 可在文案中解释为：
     - “象征天网张开，外部不利因素极强，适合研习如何在不利环境下选择退让与避开锋芒。”

## 五、在 APP 中的推荐呈现方式

1. **全局警告条**
   - 当判定为“天网四张”时：
     - 在三奇择日页顶部展示一条红色警告：
       - 标题：**“天网四张”**
       - 内容：
         - “依据《遁甲符应经》，当前为天网四张之时，万物尽伤，不宜举造百事。”
2. **格局总览中的条目**
   - 名称：**“天网四张”**
   - 分类：强凶规则
   - 作为“不可轻动”的代表格局，与“六仪击刑”等并列。
