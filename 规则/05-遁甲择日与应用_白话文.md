## 一、干支择日：五类日子吉凶速查

古人通过**日干与日支的五行生克关系**，将每日分为五类，用于军事、嫁娶、动土等大事决策。

### 1.1 五行生克基础回顾

| 关系         | 含义                              | 示例                       |
| ------------ | --------------------------------- | -------------------------- |
| **干生支**   | 天干生地支 → 能量外放，大吉       | 甲（木）生午（火）→ 甲午日 |
| **支生干**   | 地支生天干 → 能量内收，次吉       | 子（水）生甲（木）→ 甲子日 |
| **干克支**   | 天干克地支 → 主动压制，中平       | 甲（木）克戌（土）→ 甲戌日 |
| **干支同类** | 天干地支五行相同 → 和谐稳定，次吉 | 戊（土）+ 辰（土）→ 戊辰日 |
| **支克干**   | 地支克天干 → 下犯上，大凶         | 申（金）克甲（木）→ 甲申日 |

> ✅ **口诀记忆**：
> “**干生支为宝，支生干为义；干克支为制，同类为和；支克干为伐，大凶不可用。**”

------

### 1.2 五类日子详解表

| 类型             | 定义     | 吉凶  | 典型日例                                                   | 适用事项                         |
| ---------------- | -------- | ----- | ---------------------------------------------------------- | -------------------------------- |
| **宝日**（上吉） | 干生支   | ⭐⭐⭐⭐⭐ | 甲午、乙巳、丙辰、丁未、戊申、己酉、庚子、辛亥、壬寅、癸卯 | 出军、讨伐、兴造、嫁娶、百事皆宜 |
| **义日**（次吉） | 支生干   | ⭐⭐⭐⭐  | 甲子、丙寅、辛未、壬申、癸酉、庚辰、乙亥、己巳、戊午、辛丑 | 求财、合作、文书、次要事务       |
| **制日**（中平） | 干克支   | ⭐⭐⭐   | 甲戌、乙未、丙申、丁酉、戊子、己亥、庚寅、辛卯、壬午、癸巳 | 制服对手、约束下属、一般事务     |
| **和日**（次吉） | 干支同类 | ⭐⭐⭐⭐  | 甲寅、乙卯、丙午、丁巳、戊辰、己丑、庚申、辛酉、壬子、癸亥 | 和解、签约、家庭事务             |
| **伐日**（大凶） | 支克干   | ❌     | 甲申、乙酉、丙子、丁亥、戊寅、己卯、庚午、辛巳、壬辰、癸未 | **百事忌用！尤其不可出军**       |

> 📌 **重点提醒**：  
>
> - **伐日绝对不可出兵**！原文明确：“不宜出军，凶。”  
> - 宝日最吉，优先选用；义日、和日次之；制日需谨慎。

## 二、利客 vs 利主：何时先发？何时后发？

奇门不仅看日子，更看**九星加临九宫 + 季节 + 云气**，判断是“客方”（进攻方）有利，还是“主方”（防守方）有利。

> 🎯 **核心逻辑**：  
>
> - **利客** → 先举兵、高旗鸣鼓、主动出击  
> - **利主** → 后举兵、低旗衔枚、待敌而动

------

### 2.1 利客格局（客方大胜）

| 九星组合         | 适用季节             | 吉日干支           | 云气征兆                | 结论   |
| ---------------- | -------------------- | ------------------ | ----------------------- | ------ |
| 天蓬             | 秋冬                 | 壬、癸、亥、子日   | 黑色云气从**北方**来    | 客大胜 |
| 天柱、天心       | 秋 + 季夏（六月）    | 庚、辛、申、酉日   | 白色云气从**西方**来    | 客大胜 |
| 天任、天禽、天芮 | 四季月（3/6/9/12月） | 戊、己、辰戌丑未日 | 黄色云气从**东北**来    | 客大胜 |
| 天冲、天辅       | 春、冬               | 甲、乙、寅、卯日   | 青色云气从**东/东南**来 | 客胜   |
| 天英             | 春、夏               | 丙、丁、巳、午日   | 赤色云气从**南方**来    | 客大胜 |

> 💡 **操作建议**：

## 三、五阳五阴时：时辰决定行动策略

奇门认为，**一天12时辰**中，前5个（子→辰）属“阳”，后5个（巳→酉）属“阴”（注：原文以甲己日为例）。

### 3.1 五阳时（利客）

- **时间范围**：子、丑、寅、卯、辰时（约23:00–7:00）

- **天干特征**：甲、乙、丙、丁、戊

- **能量属性**：阳气上升，善神当值

- **行动策略**：

  > “先举兵，高旗鸣鼓，耀武扬兵以决胜。”

- **适宜事项**：

  - 出军征伐
  - 远行求利
  - 建国立邑
  - 拜官嫁娶
  - 百事皆吉

- **特殊效果**：逃亡者不可得（抓人易，逃跑难）

------

### 3.2 五阴时（利主）

- **时间范围**：巳、午、未、申、酉时（约9:00–17:00）

- **天干特征**：己、庚、辛、壬、癸

- **能量属性**：阴气下降，恶神当值

- **行动策略**：

  > “后举兵，低旗衔枚，待敌而后动。”

- **禁忌事项**：

  - ❌ 拜官、移徙、婚姻、出行、兴造

- **适宜事项**：

  - 画策密谋
  - 集结武备
  - 祷祀祈福

- **特殊效果**：逃亡者不可得（躲藏易，被抓难）

> 📌 **关键口诀**：
> “**得阳者，飞而不止；得阴者，伏而不起。**”

## 四、四时所利：季节决定战场选择

不同季节，天地杀气位置不同，影响作战地形选择。

| 季节       | 杀气位置                     | 作战建议                     |
| ---------- | ---------------------------- | ---------------------------- |
| **春、夏** | 杀气潜藏，阴气在下，阳气在上 | 利居**野地、低处**（避阳气） |
| **秋、冬** | 杀气在上，阴气居下           | 利居**野外高处**（顾其杀气） |

> 🌿 **原理**：
> 春夏阳盛，低处阴气可平衡；秋冬阴盛，高处可避杀气下沉。

## 五、亭亭与白奸：贵神与奸神的方位博弈

### 5.1 亭亭（天之贵神）

- **作用**：背对亭亭方位，攻击其“冲”（对面），必胜。
- **推算方法**：
  1. 用**月将**（太阳过宫）加**时辰地支**
  2. 看“神后”（子位）落在哪一宫 → 即亭亭所在

#### ▶ 实例

> 五月（月将小吉=未），加寅时 → 未+寅=申位？
> 实际：月将未加寅时，地盘寅上见未，顺数至子（神后）在**未宫** → 亭亭在**西南**（未位）
> → 背对西南，攻击东北，胜！

------

### 5.2 白奸（天之奸神）

- **作用**：需避开白奸方位，否则易中计。
- **推算方法**：
  1. 月将加时
  2. 若寅、午、戌上见**孟神**（寅申巳亥），即白奸位

#### ▶ 实例

> 正月（月将登明=亥），加时干（假设为午）→ 亥加午，登明临午
> → 白奸在**亥**（西北）
> → 行动时避开西北

> 🔁 **通用规则**：
> “**亭亭向白奸**” → 贵神指向奸神，需背亭亭、避白奸。

## 六、背孤击虚：以少胜多的战术

> “背孤击虚，一女可敌十夫。”

### 6.1 孤虚表（以旬为单位）

| 旬首 | 孤位   | 虚位（冲） |
| ---- | ------ | ---------- |
| 甲子 | 戊、亥 | 辰、巳     |
| 甲戌 | 巳、申 | 亥、寅     |
| 甲申 | 午、未 | 子、丑     |
| 甲午 | 辰、巳 | 戌、亥     |
| 甲辰 | 寅、卯 | 申、酉     |
| 甲寅 | 子、丑 | 午、未     |

> 🎯 **战术**：  
>
> - 背对“孤”方，攻击“虚”方  
> - 虚位无人防守，一击即溃

### 6.2 孤虚层级（按规模选用）

| 规模    | 用   | 说明   |
| ------- | ---- | ------ |
| 10人    | 时孤 | 最灵验 |
| 100人   | 日孤 | —      |
| 1000人  | 月孤 | —      |
| 10000人 | 年孤 | —      |

> ✅ **重点**：**时孤最验**！小规模行动优先看时辰孤虚。

## 七、实战应用技法

### 7.1 推贼兵数量

**方法**：用**月将加闻贼时辰**，看上神（地支）：

| 神将                   | 兵力数量             |
| ---------------------- | -------------------- |
| 河魁（戌）、登明（亥） | 500 / 5,000 / 50,000 |
| 太乙（巳）、登明（亥） | 400 / 4,000 / 40,000 |
| 神后（子）、胜光（午） | 600 / 6,000 / 60,000 |
| 大吉（未）、小吉（辰） | 800 / 8,000 / 80,000 |
| 功曹（寅）、传送（申） | 700 / 7,000 / 70,000 |
| 从魁（酉）、太冲（卯） | 100 / 1,000 / 10,000 |

> 🔼 **气数调整**：
>
> - 神旺（当令）→ ×10  
> - 相气（次旺）→ ×5  
> - 休气（平常）→ 原数  
> - 囚（受克）→ 减半  
> - 死（极衰）→ 再减半

------

### 7.2 迷路选择法

**场景**：行军遇三岔路，不知哪条通。

**方法**：月将加时，看“县”（即月建）落孟/仲/季：

| 月建位置           | 通路   |
| ------------------ | ------ |
| **孟**（寅申巳亥） | 左道通 |
| **仲**（子午卯酉） | 中道通 |
| **季**（辰戌丑未） | 右道通 |

> 🧭 **操作**：
> 加时后，看月建地支属性 → 选对应道路。

### 7.3 出军疫病禳解法

**问题**：军队爆发瘟疫，死者众多。

**解法**：  

> “取死人骷髅数个，闻其气即解。”

> ⚠️ **注意**：此为古法记载，**现代不可实操**！仅作历史了解。

### 7.4 兵家螺法（占卜敌情）

**工具**：黑盘 + 两只田螺

**咒语**：  

> “田螺舞，能知风雨。敌若来迫，入我城所。
> 田螺京索，风雨不着。敌若不来，各守塌廓。急急如律令。”

**观测**：

- **左螺侵右** → 我方胜  
- **右螺侵左** → 我方败  
- **互不侵犯** → 不战

> 🐌 **原理**：利用生物对磁场/湿度的敏感反应，类似“生物占卜”。

## 八、总结：择日应用心法

| 场景         | 优先看                                  | 操作要点               |
| ------------ | --------------------------------------- | ---------------------- |
| **选吉日**   | 宝日 > 义日/和日 > 制日 >> 伐日（禁用） | 查干支生克             |
| **定攻守**   | 九星+季节+云气                          | 利客则先发，利主则后发 |
| **选时辰**   | 五阳时（主动） vs 五阴时（密谋）        | 子–辰为阳，巳–酉为阴   |
| **小规模战** | 时孤虚                                  | 背孤击虚               |
| **迷路**     | 月将加时看月建                          | 孟左、仲中、季右       |
| **测敌情**   | 螺法 / 推贼数                           | 辅助判断               |

> ✅ **终极原则**：
> **遁甲择日不是迷信，而是古人融合天文、地理、气象、心理的综合决策模型。**
> 现代人可取其“顺势而为、择机而动”的智慧，用于商业、生活、风险管理。
